Markers(suffixes)






Markers(which refers to suffixes) is the main way Turkish makes its meanings(it is an agglutinative language afterall), markers are usually more logical than isolating words like in other languages.

Marker Syntax

Again, what type of vowel harmony that a marker uses is shown by its superscript number(2 = E-Type, 4 = I-type). If markers are used on a proper noun, a "'" must be added.
Everything in Turkish uses a marker: Tenses, Conjugations, Cases, etc. Things that use a marker dont need to be related to grammar.

The image above shows the positions which markers should come in: 0 is for the verb, 1 is for moods, 2 is for tenses, 3 is for personal markers and converbs, 4 is for the markers that come as a different word(Ex. "Mi:" the question marker)
