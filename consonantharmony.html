Consonant Harmony






Like vowel harmony, consonant harmony changes consonants to work with eachother. Consonants in a marker will change to work with "hard" consonants in the base word.

Consonant Harmony

In Turkish there are "hard" consonants(nothing really decides what consonants are hard). When a consonant-starting marker is added to a base word ending in a hard consonant, then the consonant is devoiced. Below are the hard consonants:

f s t k ç ş h p

If this is hard to remember, try remembering the phrase "Fıstıkçı şahap(Şahap the nut-seller)." This is useful because it contains all the hard consonants.
Only "c" and "d" are devoiced when next these consonants, other letters stay voiced.

Examples

İngiliz - English(adjective)
Türk - Turkish(adjective)

İngilizce - English(language)
Türkçe - Turkish(language)

Adam - Man
Çocuk - Child

Adamdan - From the man
Çocuktan - From the child
