Noun Cases






Noun Cases are a grammatical system where nouns change spelling(or in the case of Turkish, adds markers to them) to reflext a specific meaning or grammatical purpose. Keep in mind that while these cases can be defined a specific way, not every usage of them makes sense based on said definition.
If random capitalized symbols that do not appear in Turkish appear, reload the page.

















In Turkish, all cases have a marker for them. These case markers tell you about the state of the noun. Turkish has 6 noun cases:

  1. Accusative Definite

  2. Dative

  3. Locative

  4. Ablative

  5. Genitive

  6. Instrumental

Accusative Definite

Accusative is a little different from other languages, while it is used on the direct object of a sentence, it also requires the noun to be definite. A noun is definite if you would add "the" to it in English, or wouldn't add "bir" to it in Turkish, basically you are referring to one specific instance of the noun(s). Indefinite would be saying that you can't reference a specific instance of the noun

Examples

(Y)⁴ - Accusative case marker

Ekmek yerim - I eat bread
Ekmeği yerim - I eat the bread

Bir kedim var - I have a cat
Benim kediyi var - I have the cat

Dative

Dative is like the opposite of Accusative, it refers to the indirect object and translates to "to [x]."

Examples

(Y)² - Dative case marker

Kedileri ona veririm - I give him/her/it the cats
Şimdi ona söyle - Tell him/her now
Dative is used here because in Turkish you're saying "[subj.] is saying/talking ([obj.]) to [indirect obj.]"

Locative

This is self-explanatory, it says that someone or something is in or on the noun.

Examples

T/D² - Locative case marker

Avrupa'dayım - I'm in Europe
Mağazadayım - I'm in the store

Ablative

Ablative case expresses movement away from something, so a "from ..." meaning can be formed from this case.

Examples

T/D²n - Ablative case marker

Adamdan bir elma - An apple from the man
O Türkiye'den - He/She/It is from Türkiye(New name for Turkey, the main country that speaks Turkish).

Genitive

Genitive is used for showing possession and the subject of a relative clause. The marker for genitive is replaced with personal markers when it isnt in third-person singular. Translates to "of [x]"

Examples

(N)⁴n - Genitive case marker

Kedim - My cat
İstanbul'un insanları iyidir - Istanbul's people are good.

Instrumental

Instrumental suggests the noun is a key instrument in the action, this simply translates to "with [x]." Some say that this isn't really a case because it just uses "ile(with)" as a marker rather than a word, but for this website, it will be called a case.

Examples

(Y)l² - Instrumental case marker

Çocukla konuşuyorum - I am talking to the child
In Turkish, this verb suggests that you talk with people, not that you talk to people like söylemek
Kalemle yazmalısın - You should write with a pen
